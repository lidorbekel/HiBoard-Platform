# Hi-Board Platform
Plan, track and monitor the on-boarding process of your company's new employees

## Run the project

- Clone this repo
- run:

```shell script
npm i
```

- start:
```shell script
npm run start
```

## Run E2E tests in dev mode
```shell script
npm run e2e
```

## Run E2E tests in CI mode
```shell script
npm run e2e:ci
```
